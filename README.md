# homebrew-ggit

Homebrew tap for [ggit](https://github.com/denrou/ggit) — an interactive TUI to scan directories for git repositories and show their status.

## Install

```bash
brew install denrou/ggit/ggit
```

Or tap first:

```bash
brew tap denrou/ggit
brew install ggit
```

## Update

```bash
brew upgrade ggit
```
